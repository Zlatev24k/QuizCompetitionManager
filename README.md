# QuizCompetitionManager

## 📌 Описание

QuizCompetitionManager е уеб базирана система за управление на отборни куиз състезания. Приложението позволява организиране, провеждане и оценяване на състезания чрез автоматизирани процеси.

## 🚀 Основни функционалности

* Регистрация и вход на потребители
* Създаване и управление на отбори
* Записване за състезания
* Създаване и управление на състезания (администратор)
* Въвеждане на резултати по кръгове
* Автоматично изчисляване на класиране
* Приключване на състезания

## 🛠️ Използвани технологии

* ASP.NET Core MVC
* Entity Framework Core (Code First)
* Microsoft SQL Server
* ASP.NET Core Identity
* HTML, CSS, Razor

## 🗄️ База данни

Приложението използва SQL Server (localhost\SQLEXPRESS) и Entity Framework Core за управление на данните.

## ▶️ Стартиране на проекта

1. Отвори проекта в Visual Studio
2. Провери connection string в appsettings.json
3. Изпълни миграциите:
   Add-Migration (име на миграцията)
   Update-Database
4. Стартирай приложението (Ctrl + F5)

## 👤 Роли в системата

* Администратор – управлява състезания и резултати
* Потребител – създава отбор и участва в състезания
